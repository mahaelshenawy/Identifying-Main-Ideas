# Mastering Main Ideas - English Learning Activity

An interactive, comprehensive HTML-based learning activity designed to teach beginner to intermediate English learners how to identify and analyze main ideas, topic sentences, supporting details, and key reading comprehension concepts.

## Features

✨ **Interactive Learning Experience**
- 15 comprehensive slides with explanations and examples
- 8 challenging practice activities with immediate feedback
- Progress tracking with visual progress bar
- Responsive design for desktop and mobile devices

📚 **Comprehensive Curriculum**
- Main Idea concepts and definitions
- Topic Sentence identification
- Supporting Details recognition
- Repeated Words as clues
- Topic vs. Main Idea distinction
- Complex paragraph analysis

🎯 **Learning Objectives**
Students will be able to:
- Identify the main idea in paragraphs
- Locate topic sentences
- Recognize supporting details
- Spot repeated words that signal importance
- Distinguish between topic and main idea
- Analyze complex, multi-sentence paragraphs

## How to Use

1. **Download or Clone** this repository
2. **Open** `main_idea_advanced.html` in any web browser
3. **Click "Next"** to navigate through lessons and activities
4. **Select answers** to practice questions
5. **Receive immediate feedback** on your responses

## Content Structure

### Lessons (Slides 1-6)
- Introduction to main ideas
- What is a main idea?
- Understanding topic sentences
- Supporting details explained
- Finding clues through repeated words
- Topic vs. Main Idea comparison

### Practice Activities (Slides 7-14)
- Activity 1: Identify the Main Idea
- Activity 2: Find the Topic Sentence
- Activity 3: Identify Supporting Details
- Activity 4: Spot Repeated Words
- Activity 5: Complex Paragraph Analysis
- Activity 6: Main Idea vs. Supporting Details
- Activity 7: Identify the Topic
- Activity 8: Final Challenge (Comprehensive)

### Conclusion (Slide 15)
- Summary of learning objectives
- Encouragement for continued practice
